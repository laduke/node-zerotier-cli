# Setup
- `git pull https://github.com/laduke/node-zerotier-cli.git`
- `npm install`
- Get your ZeroTier authtoken.secret
- `ZT_AUTH=myauthtoken node index.js`
- or put `ZT_AUTH=myauthtoken` in a `.env` file
